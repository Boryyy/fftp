# Fftp - FTP/SFTP Client

Modern FTP/SFTP client with encrypted connection storage.

## Features

- FTP and SFTP support
- Encrypted connection storage with master password
- Drag-and-drop file uploads
- Transfer queue with progress tracking
- Modern, clean interface

## Installation

```bash
pip install -r requirements.txt
python main.py
```

## Usage

**Quick Connect:** Enter host, username, password, port and click Connect.

**Site Manager:** Save connections securely with master password encryption.

**Settings:** Configure themes, transfer settings, and drag-and-drop options.

## Requirements

- Python 3.8+
- PyQt6, paramiko, cryptography
